# Color Palette Picker

A responsive Color Palette Picker web application built using HTML, CSS, and JavaScript that allows users to explore, select, and copy color palettes with HEX color codes through a clean and user-friendly interface.

## Features

- Explore beautiful color palettes
- Display HEX color codes
- Copy HEX codes with a single click
- Responsive and modern user interface
- Easy to use

## Technologies Used

- HTML
- CSS
- JavaScript

## Live Demo

https://madhuri-yelure.github.io/Color-Palette-Picker/

## How to Run

1. Clone or download the repository.
2. Open the project folder.
3. Open `index.html` in your web browser.
